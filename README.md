Анализ данных сенсора (2)
ФИО: Томилов Михаил Федорович
Группа: СКБ251

1. Задание
Написать программу для анализа показаний сенсора на языке C++ с расщиренным функционалом.

2. Описание алгоритма:
  1. Основные функции программы:
     1. Функция inputData():
      принимает двойной указатель, количество дней и количество измерений
      запрашивает ввод значений от пользователя
      проверяет корректность ввода 
      очищает буфер ввода при ошибках
     2. Функция dayAverage():
      принимает указатель на строку и количество измерений
      вычисляет среднее арифметическое всех значений
      возвращает среднюю температуру за день
     3. Функция overallAverage()
      принимает двойной указатель, количество дней и измерений
      суммирует все данные измерений и делит их на кол-во измерений
      возвращает среднюю температуру по всем дням
     4. Функция showHotDays():
      вычисляет общую среднюю через overallAverage();
      для каждого дня вызывает dayAverage();
      если среднее за день > общего среднего, то выводит номер дня и его среднюю температуру
    5. Главная функция main():
      выводит заголовок программы
      запрашивает количество дней и измерений 
      выделяет динамическую память (создаёт массив указателей и массивы значений для каждого дня)
      вызывает функцию inputData() для заполнения массива
      вычисляет и выводит необходимую статистику 
      освобождает выделенную динамическую память
      завершает работу программы


3. Вывод
Написал программу на языке C++ для анализа данных сенсора. Вспомнил синтаксис языка C++, практиковался с динамической памятью (new/delete), использовал указатели и их арифметику в двумерных массивах.
